#karma
是一个单元测试用例的运行控制框架

 #安装
     npm install -g karma --save-dev
     npm install karma --save-dev
     npm install karma-jasmine --save-dev
     npm install karma-junit-reporter --save-dev
     
#配置
karma需要一个配置文件
node.js风格的

   karma init
